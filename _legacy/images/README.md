## Image Guidelines

- **Format**: JPG for photos, PNG for illustrations with transparency
- **Optimization**: Compress images for web (aim for <200KB per image)
- **Alt Text**: All images must have descriptive alt attributes in HTML
- **Responsive**: Provide multiple sizes if needed (@1x, @2x for retina displays)
- **Fallback**: Use placeholder backgrounds in CSS if images fail to load
